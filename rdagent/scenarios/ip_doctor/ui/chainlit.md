# 🏥 IP Doctor — MBS Prepayment Attribution

Welcome to **IP Doctor**, an AI-powered analyst for MBS prepayment model attribution.

Ask natural language questions about what drove your prepayment model's forecast change and get:

- 📊 **Feature attribution charts** — ranked by importance, with before/after values
- 🔢 **SMM/CPR forecast deltas** — model predictions for both periods
- 📄 **Self-contained PDF report** — narrative + all plots in one file

---

**Supported analysis types:**

| Type | Example |
|------|---------|
| Month-over-month | *"For CUSIP 3140GXPJ8, what drove the CPR change from Jan to Feb 2024?"* |
| Scenario comparison | *"Compare base vs rate_shock_100bps for CUSIPs ABC123 and DEF456."* |
